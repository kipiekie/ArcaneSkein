# Arcane Skein

Arcane Skein is a personal portfolio website built with React. It showcases my art, programming projects, and provides links to my Etsy shop, social media, and more. The site features a whimsical design inspired by the night sky and highlights my creative and technical work.

## Features

- Custom profile section with unique artwork
- Interactive navigation for Art, Programming, and About Me sections
- External links to Etsy and social platforms
- Responsive and accessible design

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ArcaneSkein.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
